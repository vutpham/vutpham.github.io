Projects

What's This RGB?

The RGB color model is based on a the science of the human eye perceives light and translates it into brain waves. However, most people simply use online color pickers and never learn the RGB color model. What's This RGB? is an interactive game to learn the RGB color model. It is designed to quickly teach the RGB color model by rapidly generating new colors after every correct guess.

Technologies: JavaScript, HTML, CSS

Artsie

Artsie is a fullstack, single-page application with a robust user authentication system. Features include CRUD, search, reviews, suggested items, and cart items.

Technologies: Ruby on Rails, JavaScript, React/Redux, PostgreSQL

Space Pong

An interactive three-dimensional pong game developed using Vanilla JavaScript and the EaselJS API. Game rendering is implemented by applying physics concepts to map the 3d graphics to the 2d plane. See if you can beat the computer! :)

Technologies: Vanilla JavaScript, EaselJS, HTML5, CSS3, HTML Canvas

GluteNo

Coming Fall 2017! GluteNo is a mobile app made to help gluten-sensitive users locate celiac-friendly restaurants. The restaurants are uploaded and reviewed by a community of users to allow gluten-sensitive users to dine out with a peace of mind.

Technologies: React Native, Django, Python, PostgreSQL
